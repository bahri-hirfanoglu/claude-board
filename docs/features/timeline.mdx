The Timeline view provides a Gantt-chart style visualization of your project's tasks, showing when each task started, how long it took, and its current status.

## Features

### Zoom Controls
Switch between three granularity levels from the toolbar:
- **Day** — Shows individual day markers
- **Week** — Shows Monday markers only
- **Month** — Shows month boundaries

### Status Swimlanes
Tasks are grouped by status (In Progress, Testing, Backlog, Done) with collapsible section headers. Click a group header to collapse/expand it.

### Weekend Shading
Saturday and Sunday columns have a subtle background highlight to distinguish weekends from workdays.

### Today Marker
A vertical blue line marks the current date, with a "TODAY" label in the header.

### Hover Tooltips
Hover over any task bar to see a rich tooltip with:
- Task title
- Start and end dates
- Duration
- Token usage
- Cost

### Task Info Panel
Each row has a left-side panel showing:
- Task type badge
- Model used
- Priority level
- Task title
- Task key
- Branch name (if any)

### Gradient Bars
Task bars use inline gradient colors matching their status:
- **Backlog** — Gray gradient
- **In Progress** — Amber gradient with shimmer animation for running tasks
- **Testing** — Claude orange gradient
- **Done** — Green gradient

### Pending Tasks
Tasks without a start date are shown in a separate "Pending" section below the timeline as a card grid.

## Interaction

- **Click** any task row or pending card to open the task detail modal
- **Collapse/expand** status groups by clicking the group header
- **Zoom** in/out using the Day/Week/Month buttons in the toolbar
