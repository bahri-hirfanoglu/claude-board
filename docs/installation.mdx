## Build from Source

Building from source requires both **Rust** and **Node.js** toolchains.

### Prerequisites

- [Rust](https://rustup.rs/) (stable toolchain)
- [Node.js](https://nodejs.org) >= 18.0.0
- Platform-specific dependencies for Tauri — see the [Tauri v2 prerequisites](https://v2.tauri.app/start/prerequisites/)

### Steps

```bash
git clone https://github.com/bahri-hirfanoglu/claude-board.git
cd claude-board
npm install
npx tauri dev
```

`npx tauri dev` builds the Rust backend, starts the Vite dev server for the React frontend, and opens the app window with hot-reload enabled.

## Production Build

To produce a distributable installer:

```bash
npx tauri build
```

The output is written to `src-tauri/target/release/bundle/` and includes platform-specific installers (`.exe`/`.msi` on Windows, `.dmg` on macOS, `.deb`/`.AppImage` on Linux).
