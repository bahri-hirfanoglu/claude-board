## Prerequisites

- [Claude Code CLI](https://docs.anthropic.com/en/docs/claude-code) installed and authenticated
- [Node.js](https://nodejs.org) >= 18.0.0 (only for source install)

## Install from Source

<Tabs>
  <Tab title="npm">
    ```bash
    git clone https://github.com/bahri-hirfanoglu/claude-board.git
    cd claude-board
    npm run setup
    npm run dev
    ```
  </Tab>
  <Tab title="Docker">
    ```bash
    docker build -t claude-board .
    docker run -p 4000:4000 -v claude-data:/app/data claude-board
    ```
  </Tab>
  <Tab title="Docker Compose">
    ```bash
    docker compose up -d
    ```
  </Tab>
</Tabs>

Open [http://localhost:4000](http://localhost:4000) in your browser.

## Production

```bash
npm run build
npm start
```

The server starts on port `4000` by default. Set the `PORT` environment variable to change it.
