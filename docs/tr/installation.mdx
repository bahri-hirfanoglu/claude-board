## Onkosuller

Claude Board'u kurmadan once asagidakilerin hazir oldugundan emin olun:

<Steps>
  <Step title="Claude Code CLI" icon="terminal">
    [Claude Code CLI](https://docs.anthropic.com/en/docs/claude-code) aracini kurun ve kimlik dogrulamasi yapin. Dogrulamak icin `claude --version` komutunu calistirin.
  </Step>
  <Step title="Git" icon="code-branch">
    Dal otomasyonu ve commit takibi icin Git kurulu olmalidir. Dogrulamak icin `git --version` komutunu calistirin.
  </Step>
  <Step title="GitHub CLI (istege bagli)" icon="github">
    [Otomatik PR](/features/auto-pr) olusturma ozelligini kullanmak istiyorsaniz [`gh`](https://cli.github.com/) aracini kurun ve `gh auth login` komutunu calistirin.
  </Step>
</Steps>

### Ilk Calistirma

Ilk calistirmada [6 adimli kurulum sihirbazi](/desktop/setup) sizi su adimlarda yonlendirir:

1. **Dil secimi** — Ingilizce veya Turkce
2. **Sistem kontrolu** — Claude CLI, Node.js, Git ve port uygunlugunu dogrular
3. **Veri dizini** — SQLite veritabani ve yuklemelerin depolandigi konum
4. **MCP sunucu portu** — Varsayilan `4000`, uygunluk otomatik kontrol edilir
5. **Ilk proje** — Calisma dizini ile birlikte ilk projenizi olusturur

<Tip>Kurulum sihirbazi yalnizca bir kez calisir. Ayarlari daha sonra uygulama menusunden degistirebilirsiniz.</Tip>

## Kaynak Koddan Derleme

Kaynak koddan derleme icin hem **Rust** hem de **Node.js** arac zincirleri gereklidir.

### Onkosuller

| Arac | Surum | Amac |
|------|-------|------|
| [Rust](https://rustup.rs/) | Stable | Tauri v2 arka ucu |
| [Node.js](https://nodejs.org) | >= 18.0.0 | React on yuzu ve derleme araclari |
| Platform bagimliliklari | En son | [Tauri v2 onkosullar](https://v2.tauri.app/start/prerequisites/) |

### Adimlar

```bash
git clone https://github.com/bahri-hirfanoglu/claude-board.git
cd claude-board
npm install
npx tauri dev
```

`npx tauri dev` komutu Rust arka ucunu derler, React on yuzu icin Vite gelistirme sunucusunu baslatir ve anlik yenileme etkin sekilde uygulama penceresini acar.

## Uretim Derlemesi

Dagitilabilir bir yukleyici olusturmak icin:

```bash
npx tauri build
```

Cikti `src-tauri/target/release/bundle/` dizinine yazilir ve platforma ozel yukleyiciler icerir.

| Platform | Cikti |
|----------|-------|
| Windows | `bundle/msi/` icinde `.exe` ve `.msi` |
| macOS | `bundle/dmg/` icinde `.dmg` |
| Linux | `bundle/deb/` ve `bundle/appimage/` icinde `.deb` ve `.AppImage` |

## Kurulumu Dogrulama

Kurulumdan sonra her seyin duzgun calistigini dogrulayin:

<Accordion title="Dogrulama kontrol listesi">
  1. Claude Board'u baslatin — kurulum sihirbazini (ilk calistirma) veya ana panoyu gormelisiniz
  2. Ayarlar'i acin ve MCP sunucu portunun aktif oldugunu onaylayin
  3. Herhangi bir kod dizinine isaret eden bir test projesi olusturun
  4. Basit bir gorev olusturun (ornegin "Proje kok dizinindeki tum dosyalari listele") ve Devam Ediyor durumuna tasayin
  5. Canli terminali izleyin — Claude'un ciktisini gercek zamanli olarak gormalisiniz
</Accordion>
