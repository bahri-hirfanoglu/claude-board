## Önkoşullar

Claude Board'u kurmadan önce aşağıdakilerin hazır olduğundan emin olun:

<Steps>
  <Step title="Claude Code CLI" icon="terminal">
    [Claude Code CLI](https://docs.anthropic.com/en/docs/claude-code) aracını kurun ve kimlik doğrulaması yapın. Doğrulamak için `claude --version` komutunu çalıştırın.
  </Step>
  <Step title="Git" icon="code-branch">
    Dal otomasyonu ve commit takibi için Git kurulu olmalıdır. Doğrulamak için `git --version` komutunu çalıştırın.
  </Step>
  <Step title="GitHub CLI (isteğe bağlı)" icon="github">
    [Otomatik PR](/features/auto-pr) oluşturma özelliğini kullanmak istiyorsanız [`gh`](https://cli.github.com/) aracını kurun ve `gh auth login` komutunu çalıştırın.
  </Step>
</Steps>

### İlk Çalıştırma

İlk çalıştırmada [6 adımlı kurulum sihirbazı](/desktop/setup) sizi şu adımlarda yönlendirir:

1. **Dil seçimi** — İngilizce veya Türkçe
2. **Sistem kontrolü** — Claude CLI, Node.js, Git ve port uygunluğunu doğrular
3. **Veri dizini** — SQLite veritabanı ve yüklemelerin depolandığı konum
4. **MCP sunucu portu** — Varsayılan `4000`, uygunluk otomatik kontrol edilir
5. **İlk proje** — Çalışma dizini ile birlikte ilk projenizi oluşturur

<Tip>Kurulum sihirbazı yalnızca bir kez çalışır. Ayarları daha sonra uygulama menüsünden değiştirebilirsiniz.</Tip>

## Kaynak Koddan Derleme

Kaynak koddan derleme için hem **Rust** hem de **Node.js** araç zincirleri gereklidir.

### Önkoşullar

| Araç | Sürüm | Amaç |
|------|-------|------|
| [Rust](https://rustup.rs/) | Stable | Tauri v2 arka ucu |
| [Node.js](https://nodejs.org) | >= 18.0.0 | React ön yüzü ve derleme araçları |
| Platform bağımlılıkları | En son | [Tauri v2 önkoşullar](https://v2.tauri.app/start/prerequisites/) |

### Adımlar

```bash
git clone https://github.com/bahri-hirfanoglu/claude-board.git
cd claude-board
npm install
npx tauri dev
```

`npx tauri dev` komutu Rust arka ucunu derler, React ön yüzü için Vite geliştirme sunucusunu başlatır ve anlık yenileme etkin şekilde uygulama penceresini açar.

## Üretim Derlemesi

Dağıtılabilir bir yükleyici oluşturmak için:

```bash
npx tauri build
```

Çıktı `src-tauri/target/release/bundle/` dizinine yazılır ve platforma özel yükleyiciler içerir.

| Platform | Çıktı |
|----------|-------|
| Windows | `bundle/msi/` içinde `.exe` ve `.msi` |
| macOS | `bundle/dmg/` içinde `.dmg` |
| Linux | `bundle/deb/` ve `bundle/appimage/` içinde `.deb` ve `.AppImage` |

## Kurulumu Doğrulama

Kurulumdan sonra her şeyin düzgün çalıştığını doğrulayın:

<Accordion title="Doğrulama kontrol listesi">
  1. Claude Board'u başlatın — kurulum sihirbazını (ilk çalıştırma) veya ana panoyu görmelisiniz
  2. Ayarlar'ı açın ve MCP sunucu portunun aktif olduğunu onaylayın
  3. Herhangi bir kod dizinine işaret eden bir test projesi oluşturun
  4. Basit bir görev oluşturun (örneğin "Proje kök dizinindeki tüm dosyaları listele") ve Devam Ediyor durumuna taşıyın
  5. Canlı terminali izleyin — Claude'un çıktısını gerçek zamanlı olarak görmelisiniz
</Accordion>
