Analitik görünümü kapsamlı maliyet takibi, model karşılaştırması, verimlilik metrikleri ve sıralanabilir görev performans tablosu sunar. Board araç çubuğundan (TrendingUp simgesi) erişebilirsiniz.

## Maliyet Özeti

Üstteki dört istatistik kartı şu bilgileri gösterir:

| Metrik | Açıklama |
|--------|----------|
| **Toplam Maliyet** | Tüm görevler genelindeki kümülatif API maliyeti |
| **Toplam Token** | Giriş + çıkış tokenları ve dağılımı |
| **Ort. Maliyet/Görev** | Kullanım ile birlikte görev başına ortalama maliyet |
| **Ort. Token/Görev** | Ortalama token tüketimi |

## Model Karşılaştırması

Modelleri maliyet ve token kullanımına göre karşılaştıran çubuk grafik:

- Her model (Haiku, Sonnet, Opus) görev sayısını, toplam tokenleri ve toplam maliyeti gösterir
- Modele göre renk kodlu: yeşil (Haiku), mavi (Sonnet), mor (Opus)
- İş yükleriniz için hangi modelin en uygun maliyetli olduğunu belirlemenize yardımcı olur

## Maliyet Trendi

Zaman içindeki harcamayı gösteren günlük maliyet grafiği. Günlük kesin değerler (maliyet, token, görev sayısı) için üzerine gelin.

## Verimlilik Metrikleri

Dört verimlilik kartı:

| Metrik | Açıklama |
|--------|----------|
| **İş Hacmi** | Çalışma süresi başına tamamlanan görev sayısı |
| **Başarı Oranı** | Toplama kıyasla done/testing durumuna ulaşan görevlerin yüzdesi |
| **Önbellek Oranı** | Prompt önbellek isabet yüzdesi (yüksekse = daha ucuz) |
| **Ort. Tur Sayısı** | Görev başına ortalama konuşma tur sayısı |

## Görev Performans Tablosu

Kullanım verileriyle birlikte tüm görevlerin sıralanabilir tablosu:

- **Sıralama**: Maliyet (varsayılan), Token veya Süre — sütun başlıklarını tıklayın
- **Sütunlar**: Görev anahtarı, başlık, model, süre, token, maliyet
- **En pahalı 3 görev** uyarı simgesiyle vurgulanır
- 20 satır sayfalandırmalı

<Tip>Pahalı görevleri belirlemek için maliyete göre azalan sıralama yapın. Dokümantasyon veya bakış görevleri gibi basit işler için daha ucuz bir model (Haiku) kullanmayı düşünün.</Tip>
