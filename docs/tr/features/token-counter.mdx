Token sayacı, görev açıklamanızı yazarken tahmini token sayısını ve giriş maliyetini gösterir. Bir görevi çalıştırmadan önce API maliyetlerini tahmin etmenize yardımcı olur.

## Nasıl Çalışır

Görev oluşturma penceresinde yazarken sayaç şu hesaplamaları yapar:

- **Token tahmini** — Karakter sayısına dayalı yaklaşık token sayısı (~4 karakter/token)
- **Giriş maliyeti** — Seçilen modelin fiyatlandırmasına göre tahmini maliyet
- **Karakter sayısı** — Prompt'unuzun ham karakter uzunluğu

Sayaç, içerik 10 tokeni aştıktan sonra açıklama alanının altında görünür.

## Model Fiyatlandırması

Maliyet tahminleri, görev seçeneklerinde seçilen modele dayanır:

| Model | Giriş Maliyeti (1M token başına) |
|-------|----------------------------------|
| **Haiku** | $0.25 |
| **Sonnet** | $3.00 |
| **Opus** | $15.00 |

<Info>Bunlar yalnızca giriş token tahminleridir. Gerçek görev maliyeti, görev karmaşıklığına göre değişen çıkış tokenleri, önbellek tokenleri ve araç kullanımını içerir.</Info>

<Tip>Aşırı uzun prompt'ları tespit etmek için token sayacını kullanın. Kısa ve odaklı bir prompt, genellikle uzun olandan daha iyi sonuçlar üretir.</Tip>
