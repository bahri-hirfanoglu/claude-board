Uygulama Ayarları, Claude Board için merkezi bir yapılandırma paneli sağlar. Panoya **dişli simgesinden** veya proje açılır menüsündeki **Uygulama Ayarları** girdisinden erişin.

## Ayarlara Erişim

<Steps>
  <Step title="Panodan" icon="grid-2">
    Pano başlığının sağ üst köşesindeki, dil seçicisinin yanındaki dişli simgesine tıklayın.
  </Step>
  <Step title="Proje Görünümünden" icon="chevron-down">
    Proje açılır menüsünü açın ve 3x3 ayarlar ızgarasında **App Settings** seçeneğine tıklayın.
  </Step>
</Steps>

## Genel Ayarlar

### Başlangıçta Aç

Bilgisayarınız başladığında Claude Board'u otomatik olarak açın. İşletim sisteminin yerel otomatik başlangıç mekanizmasını kullanır:

| Platform | Mekanizma |
|----------|-----------|
| **Windows** | Kayıt defteri başlangıç girdisi |
| **macOS** | LaunchAgent |

<Info>Bu ayar yalnızca Tauri masaüstü uygulamasında kullanılabilir, web modunda değil.</Info>

### Tepsiye Küçült

Etkinleştirildiğinde, pencereyi kapatmak Claude Board'u çıkış yapmak yerine **sistem tepsisine** gizler. Uygulama arka planda çalışmaya devam eder ve şu şekillerde geri yüklenebilir:

- Tepsi simgesine **tıkla** (sol tıklama)
- **Sağ tıklama** → Tepsi bağlam menüsünden **Claude Board**
- Tamamen çıkmak için: **Sağ tıklama** → **Quit**

<Warning>Bu yalnızca masaüstü uygulamasına özel bir özelliktir. Uygulama çalışırken sistem tepsisi simgesi her zaman görünürdür.</Warning>

### Silmeden Önce Onayla

Görevleri veya projeleri silmeden önce bir onay penceresi göster. Varsayılan olarak etkindir.

### Terminali Otomatik Aç

Bir görev çalıştığında terminal panelini otomatik olarak aç. Terminali manuel olarak açmayı tercih ediyorsanız devre dışı bırakın.

### Varsayılan Model

Yeni görevler için varsayılan Claude modelini ayarlayın:

| Model | Açıklama |
|-------|----------|
| **Haiku** | Hızlı ve hafif |
| **Sonnet** | Dengeli (varsayılan) |
| **Opus** | En yetenekli |

### Varsayılan Düşünme Eforu

Yeni görevler için varsayılan düşünme eforu seviyesini ayarlayın:

| Seviye | Açıklama |
|--------|----------|
| **Low** | Hızlı, yüzeysel analiz |
| **Medium** | Dengeli derinlik (varsayılan) |
| **High** | Derin akıl yürütme ve keşfetme |

### Dil

Uygulama görünüm dilini değiştirin. Şu anda desteklenen diller:

- **English**
- **Turkish** (Türkçe)

Dil değişikliği, bildirim mesajları dahil tüm arayüz öğelerinde anında yürürlüğe girer.

## Ayar Depolama

Ayarlar yerel SQLite veritabanında (`app_settings` tablosu) anahtar-değer çiftleri olarak saklanır. Değişiklikler değiştirildiğinde anında kaydedilir — ayrı bir "Kaydet" butonu yoktur.

## API

Ayarlara API üzerinden erişilebilir:

```
GET  /api/settings          # Tüm ayarları getir
PUT  /api/settings          # Ayarları güncelle (kısmi güncelleme)
```

Tauri masaüstü için:
```javascript
api.getAppSettings()
api.updateAppSettings({ key: value })
```
