İzin modları, Claude'un sisteminizle nasıl etkileştiğini belirler. Claude Board `--no-input` (etkileşimli istem yok) ile çalıştığı için, izin yapılandırması kritik öneme sahiptir.

## İzin Modları

<Tabs>
  <Tab title="Auto Accept">
    Claude izin istemeden **tüm** araçları kullanabilir. Bu en verimli moddur — Claude onay bekleyerek takılmaz.

    ```
    claude --no-input --dangerously-skip-permissions
    ```

    <Warning>Bu, Claude'a proje dizininizdeki dosyaları okuma, yazma ve komut çalıştırma için tam erişim verir. Yalnızca güvenilir kod tabanları veya yalıtılmış ortamlarda kullanın.</Warning>
  </Tab>
  <Tab title="Allowed Tools">
    Yalnızca belirlediğiniz araç kategorilerine izin verilir. Claude bunları serbestçe kullanabilir ancak başka hiçbir şeye erişemez.

    ```
    claude --no-input --allowedTools "Read,Edit,Write,Bash"
    ```

    Bu iyi bir orta yoldur — Claude'u otonom tutarken etki alanını siz kontrol edersiniz.
  </Tab>
  <Tab title="Default">
    Claude CLI'ın yerleşik izin sistemini kullanır. Etkileşimli onay gerektiren bir araca ihtiyaç duyulursa ajanların takılmasına neden olabilir.

    <Note>`--no-input` etkileşimli izin istemlerini engellediğinden Claude Board için önerilmez.</Note>
  </Tab>
</Tabs>

## Araç Kategorileri

Allowed Tools modunu kullanırken, herhangi bir kombinasyona izin verebilirsiniz:

| Araç | Ne Yapar |
|------|----------|
| `Read` | Diskten dosya içeriklerini okur |
| `Edit` | Mevcut dosyaları değiştirir |
| `Write` | Yeni dosyalar oluşturur |
| `Bash` | Kabuk komutları çalıştırır |
| `Glob` | Desene göre dosya arar |
| `Grep` | Regex ile dosya içeriği arar |

## Öneriler

<AccordionGroup>
  <Accordion title="Başlangıç">
    Ajanların takılmasını önlemek için **Auto Accept** kullanın. Sonuçları görmenin en hızlı yoludur.
  </Accordion>
  <Accordion title="Üretim projeleri">
    Açık kontrol ile tam işlevsellik için `Read,Edit,Write,Bash,Glob,Grep` ile birlikte **Allowed Tools** kullanın.
  </Accordion>
  <Accordion title="Salt okunur analiz">
    Claude'un değişiklik yapmadan kodu analiz etmesini istiyorsanız yalnızca `Read,Glob,Grep` ile **Allowed Tools** kullanın.
  </Accordion>
</AccordionGroup>
