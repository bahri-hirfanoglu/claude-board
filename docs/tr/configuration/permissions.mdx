Izin modlari, Claude'un sisteminizle nasil etkilestigini belirler. Claude Board `--no-input` (etkilesimli istem yok) ile calistigi icin, izin yapilandirmasi kritik oneme sahiptir.

## Izin Modlari

<Tabs>
  <Tab title="Auto Accept">
    Claude izin istemeden **tum** araclari kullanabilir. Bu en verimli moddur — Claude onay bekleyerek takilmaz.

    ```
    claude --no-input --dangerously-skip-permissions
    ```

    <Warning>Bu, Claude'a proje dizininizdeki dosyalari okuma, yazma ve komut calistirma icin tam erisim verir. Yalnizca guvenilir kod tabanlari veya yalitilmis ortamlarda kullanin.</Warning>
  </Tab>
  <Tab title="Allowed Tools">
    Yalnizca belirlediginiz arac kategorilerine izin verilir. Claude bunlari serbestce kullanabilir ancak baska hicbir seye erisemez.

    ```
    claude --no-input --allowedTools "Read,Edit,Write,Bash"
    ```

    Bu iyi bir orta yoldur — Claude'u otonom tutarken etki alanini siz kontrol edersiniz.
  </Tab>
  <Tab title="Default">
    Claude CLI'in yerlesik izin sistemini kullanir. Etkilesimli onay gerektiren bir araca ihtiyac duyulursa ajanlarin takilmasina neden olabilir.

    <Note>`--no-input` etkilesimli izin istemlerini engellediginden Claude Board icin onerilmez.</Note>
  </Tab>
</Tabs>

## Arac Kategorileri

Allowed Tools modunu kullanirken, herhangi bir kombinasyona izin verebilirsiniz:

| Arac | Ne Yapar |
|------|----------|
| `Read` | Diskten dosya iceriklerini okur |
| `Edit` | Mevcut dosyalari degistirir |
| `Write` | Yeni dosyalar olusturur |
| `Bash` | Kabuk komutlari calistirir |
| `Glob` | Desene gore dosya arar |
| `Grep` | Regex ile dosya icerigi arar |

## Oneriler

<AccordionGroup>
  <Accordion title="Baslangic">
    Ajanlarin takilmasini onlemek icin **Auto Accept** kullanin. Sonuclari gormenin en hizli yoludur.
  </Accordion>
  <Accordion title="Uretim projeleri">
    Acik kontrol ile tam islevsellik icin `Read,Edit,Write,Bash,Glob,Grep` ile birlikte **Allowed Tools** kullanin.
  </Accordion>
  <Accordion title="Salt okunur analiz">
    Claude'un degisiklik yapmadan kodu analiz etmesini istiyorsaniz yalnizca `Read,Glob,Grep` ile **Allowed Tools** kullanin.
  </Accordion>
</AccordionGroup>
