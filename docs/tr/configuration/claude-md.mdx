`CLAUDE.md`, Claude'un bir proje üzerinde çalışmaya başladığında otomatik olarak okuduğu özel bir dosyadır. Projeye özel talimatlar, kurallar ve bağlam bilgisi içerir. Claude Board bu dosyayı yönetmek için yerleşik bir düzenleyici içerir.

## CLAUDE.md Nedir?

Projenizin kökünde bulunan bir `CLAUDE.md` dosyası Claude'a şunları söyler:

- Projenin nasıl yapılandırıldığını
- Hangi kurallara uyulması gerektiğini
- Derleme ve test için hangi komutların kullanılacağını
- Önemli mimari kararları

```markdown
# Project: My API

## Stack
- Node.js + Express + TypeScript
- PostgreSQL with Prisma ORM
- Jest for testing

## Commands
- `npm run build` — compile TypeScript
- `npm test` — run all tests
- `npm run lint` — check code style

## Conventions
- Use Prisma for all database access
- All endpoints return JSON with `{ data, error }` shape
- Write integration tests for every new endpoint
```

## Yerleşik Düzenleyici

Claude Board, `CLAUDE.md` dosyasını görüntülemek ve düzenlemek için proje ayarlarında bir metin düzenleyici sunar. Değişiklikler doğrudan projenizin çalışma dizinindeki dosyaya kaydedilir.

<Steps>
  <Step title="Proje ayarlarını açın">
    Proje panonuzdaki dişli simgesine tıklayın.
  </Step>
  <Step title="CLAUDE.md sekmesine gidin">
    CLAUDE.md bölümünü seçin.
  </Step>
  <Step title="Düzenleyin ve kaydedin">
    İçeriği değiştirin ve Kaydet'e tıklayın. Dosya projenizin çalışma dizinine yazılır.
  </Step>
</Steps>

## CLAUDE.md ile Bağlam Parçaları Karşılaştırması

| Özellik | CLAUDE.md | Bağlam Parçaları |
|---------|-----------|-----------------|
| **Konum** | Deponuzdaki dosya | Claude Board'un veritabanında saklanır |
| **Sürümlendirme** | Evet, git'e commit edilir | Hayır |
| **Açma/Kapama** | Her zaman aktif | Tek tek etkinleştirilebilir/devre dışı bırakılabilir |
| **Kapsam** | Claude CLI tarafından doğrudan okunur | Claude Board tarafından prompt'a enjekte edilir |

<Tip>Depoda kalması gereken kararlı proje kuralları için `CLAUDE.md` kullanın. Sık değişen geçici kurallar veya talimatlar için bağlam parçaları kullanın.</Tip>
