Claude Board yapılandırmasını uygulama veri dizini içindeki bir `config.json` dosyasında saklar. Ayarlar, ilk başlatmada [kurulum sihirbazı](/desktop/setup) veya uygulama içi ayarlar menüsü aracılığıyla da değiştirilebilir.

## Yapılandırma Dosyası Konumu

| Platform | Varsayılan Yol |
|----------|---------------|
| Windows | `%APPDATA%/claude-board/config.json` |
| macOS | `~/Library/Application Support/claude-board/config.json` |
| Linux | `~/.config/claude-board/config.json` |

## Mevcut Ayarlar

| Ayar | Varsayılan | Açıklama |
|------|-----------|----------|
| `port` | `4000` | Claude runner entegrasyonu için MCP HTTP sunucu portu |
| `dataDir` | İşletim sistemine özel | Veritabanı ve yüklemeler için dizin |

## MCP Sunucu Portu

Yerleşik MCP HTTP sunucusu, Claude Code runner'larının Claude Board ile iletişim kurmasını sağlar. Portu `config.json` içinde ayarlayın:

```json
{
  "port": 8080
}
```

<Note>Port kullanımda ise MCP sunucusu başlatılmaz. Mevcut bir port seçin veya çakışan işlemi durdurun.</Note>

## Veri Dizini

Claude Board'un SQLite veritabanını ve yüklenen dosya eklerini nerede saklayacağını kontrol eder:

```json
{
  "dataDir": "/path/to/data"
}
```

Platforma göre varsayılan konumlar:

| Platform | Varsayılan Yol |
|----------|---------------|
| Windows | `%APPDATA%/claude-board/` |
| macOS | `~/Library/Application Support/claude-board/` |
| Linux | `~/.config/claude-board/` |

Dizin şu öğeleri içerir:

- `claude-board.db` — SQLite veritabanı (tüm projeler, görevler, ayarlar)
- `uploads/` — dosya ekleri
- `config.json` — uygulama yapılandırması

<Warning>Veri dizinini düzenli olarak yedekleyin. Tüm proje verilerinizi, görev geçmişinizi ve günlüklerinizi içerir.</Warning>

## Ortam Değişkenleri

Gelişmiş kullanım senaryoları için aşağıdaki ortam değişkenleri hala geçerlidir ve `config.json` değerlerini geçersiz kılar:

| Değişken | Açıklama |
|----------|----------|
| `CLAUDE_BOARD_DATA` | Veri dizini yolunu geçersiz kılar |
| `PORT` | MCP sunucu portunu geçersiz kılar |
