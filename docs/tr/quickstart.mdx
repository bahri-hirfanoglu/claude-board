Bu rehber ilk projenizi oluşturma, bir görev çalıştırma ve sonucu inceleme adımlarında size yol gösterir. 5 dakikadan kısa sürede bir yapay zeka ajanının kod yazmasını sağlayacaksınız.

## 1. Proje oluşturun

<Steps>
  <Step title="Claude Board'u açın">
    Masaüstü uygulamasını başlatın. İlk kez çalıştırıyorsanız kurulum sihirbazı sizi ilk yapılandırma boyunca yönlendirecektir.
  </Step>
  <Step title="'Yeni Proje' butonuna tıklayın">
    Proje adı, slug ve **çalışma dizini** girin — Claude'un kod okuyup yazacağı yerel dosya yolu.

    <Tip>Çalışma dizinini mevcut bir depoya yönlendirin, en iyi sonuçları böyle alırsınız. Claude mevcut kod yapınızı okuyabildiğinde çok daha verimli çalışır.</Tip>
  </Step>
  <Step title="İzinleri yapılandırın">
    Bir izin modu seçin:

    | Mod | Kime uygun | Açıklama |
    |-----|------------|----------|
    | **Auto Accept** | Çoğu kullanıcı | Claude tüm araçları sormadan kullanır |
    | **Allowed Tools** | Güvenlik odaklı | Yalnızca belirtilen araç kategorileri |
    | **Default** | Özel yapılandırmalar | Claude'un yerleşik ayarlarını kullanır |

    <Warning>Claude `--no-input` parametresiyle çalışır, bu nedenle etkileşimli olarak izin isteyemez. En iyi sonuçlar için "Auto Accept" modunu kullanın.</Warning>
  </Step>
</Steps>

## 2. Görev oluşturun ve çalıştırın

<Steps>
  <Step title="'Yeni Görev' butonuna tıklayın" icon="plus">
    Açık ve ayrıntılı bir tanım yazın. Ne kadar çok detay verirseniz Claude o kadar iyi performans gösterir.

    <Accordion title="Örnek: İyi ve belirsiz görev tanımları">
      **İyi:**
      ```
      Title: Add JWT authentication middleware
      Description: Implement login/register endpoints with JWT tokens.
      Use bcrypt for password hashing. Add middleware for protected routes.
      Follow the existing Express.js patterns in src/routes/.
      ```

      **Çok belirsiz:**
      ```
      Title: Add auth
      Description: Make authentication work
      ```
    </Accordion>
  </Step>
  <Step title="Görevi yapılandırın" icon="sliders">
    | Ayar | Seçenekler | Ne zaman kullanılmalı |
    |------|-----------|----------------------|
    | **Model** | `opus` / `sonnet` / `haiku` | Karmaşık işler için Opus, dengeli işler için Sonnet, basit işler için Haiku |
    | **Tür** | feature / bugfix / refactor / docs / test / chore | Görevi kategorize eder ve [prompt şablonları](/features/prompt-templates) ile eşleştirir |
    | **Öncelik** | low / medium / high / urgent | [Kuyrük sıralamasını](/features/queue) etkiler |
  </Step>
  <Step title="Görevi başlatın" icon="play">
    Görevi **Devam Ediyor** durumuna sürükleyin veya sağ tıklayıp "Devam Ediyor" seçeneğini seçin. Claude anında başlar, bir dal oluşturur ve kodlamaya geçer.
  </Step>
  <Step title="Canlı terminali izleyin" icon="terminal">
    Görev kartındaki terminal simgesine tıklayarak Claude'un gerçek zamanlı çalışmasını izleyin. Şu bilgileri göreceksiniz:
    - **Read** — Claude'un incelediği dosyalar
    - **Write/Edit** — yapılan kod değişiklikleri
    - **Bash** — çalıştırılan komutlar (testler, git vb.)
  </Step>
  <Step title="Sonucu inceleyin" icon="check">
    Claude tamamladığında görev **Test** aşamasına geçer. İnceleme butonuna tıklayarak:
    - **Onayla** — Tamamlandı durumuna taşır, isteğe bağlı olarak PR oluşturur
    - **Değişiklik İste** — geri bildirim verin, Claude otomatik olarak üzerinde çalışır
  </Step>
</Steps>
