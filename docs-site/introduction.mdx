# Welcome to Claude Board

Claude Board is an open-source desktop application that turns your local machine into an AI-powered development squad. Create tasks on a visual Kanban board, and Claude agents autonomously write code, create branches, and commit changes — all while you watch in real-time.

<Frame>
  <img src="/images/dashboard.png" alt="Claude Board Dashboard" />
</Frame>

## How it works

<Steps>
  <Step title="Create a task" icon="plus">
    Describe what you need — a feature, bugfix, refactor, or test. Pick a model (Opus, Sonnet, or Haiku) and set the priority.
  </Step>
  <Step title="Claude starts coding" icon="play">
    Drag to "In Progress" and a Claude agent spawns automatically. It reads your codebase, writes code, runs tests, and commits — visible in the live terminal.
  </Step>
  <Step title="Review and ship" icon="check">
    The task moves to "Testing". Review the changes, approve or request revisions. Claude picks up your feedback and iterates.
  </Step>
</Steps>
