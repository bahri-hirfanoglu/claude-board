The board toolbar includes model filter chips that let you quickly focus on tasks assigned to a specific Claude model.

## Usage

Filter chips appear in the toolbar next to the view toggle buttons when your project has tasks using more than one model. Each chip shows:

- A **colored dot** matching the model's color (green for Haiku, blue for Sonnet, purple for Opus)
- The **model name**
- The **task count** for that model

Click a chip to filter. Click again or press the **X** button to clear the filter.

## Scope

The model filter applies across **all views**:

- **Board view** — Columns show only filtered tasks
- **List view** — Table rows are filtered
- **Timeline view** — Only filtered tasks appear on the timeline
- **Summary view** — Statistics reflect filtered tasks only

## Model Colors

| Model | Color | Dot |
|-------|-------|-----|
| Haiku | Green | `#4ade80` |
| Sonnet | Blue | `#60a5fa` |
| Opus | Purple | `#c084fc` |
